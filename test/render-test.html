📋 Checklist for live site debugging
  1. Open https://jubo-line-badminton.netlify.app/tournament-ui-flow
  2. Open DevTools → Console tab
  3. Sign in with Google (top-right)
  4. After sign-in, check console for [pc] messages
  5. Click 💬 留言模式 button
  6. Click anywhere on the phone screen
  7. Type a comment and submit
  8. Check console: should see [pc] addDoc x=XX y=XX
  9. Check console: should see [pc] snapshot N comments
  10. Check console: should see [pc] renderAnnotations N annotations

If any step is missing from console, that's where the bug is.
